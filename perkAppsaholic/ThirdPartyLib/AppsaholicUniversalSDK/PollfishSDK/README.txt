Pollfish SDK for iOS
------------------------

The is the Pollfish SDK for iOS.

Requirements:
- A Pollfish API Key for your app (http://www.pollfish.com)
- Pollfish will be shown only on +6.0 iOS, lives well with <6.0 though
- Please see documentation for handling view hierarchy changes in app lifecycle

The SDK allows the completion of surveys through a Pollfish enabled app

More information can be found in the documentation online at https://www.pollfish.com/ios
